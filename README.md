# Python Event Calendar

Object-oriented Python application for managing calendar events.
The project focuses on clean code structure, separation of concerns
and basic data persistence.

## Features
- Event creation with date, time, duration and title
- Conversion and handling of datetime objects
- Filtering events by month or by date
- Controller-based application logic
- Saving and loading events from storage

## Project Structure
- `Event`: represents a calendar event and related date logic
- `Controller`: manages application logic
- `CalendarDisplay`: handles event display
- `SaveSystem`: responsible for saving and loading data

## Technologies
- Python 3
- Object-Oriented Programming (OOP)
- Datetime module

## How to Run
1. Make sure Python 3 is installed
2. Run the main script:
python main.py
